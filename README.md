# mitchell-jordan-portfolio
Software development portfolio featuring selected projects in full-stack development, problem-solving, and system design.

In this repo, I am covering the projects I am most proud of and the projects that taught me the most over my time in Durham College's Computer Programming and Analysis Program.

### Bakery Buddy

Bakery Buddy is a full-stack inventory and production management system designed to simulate real-world bakery operations, including ingredient tracking, recipe management, and batch production workflows.
The application focuses on maintaining accurate inventory levels while supporting production processes such as baking batches and updating ingredient usage in real time.
I contributed to both the system design and backend development, where I implemented core database logic and API functionality. This included designing relational schemas, handling inventory transactions, and ensuring data consistency across operations such as batch production and sales.
Through this project, I strengthened my understanding of transactional systems, database design, and backend architecture, particularly in managing state changes across interconnected processes.

### Cloud-Deployed-Bookstore with CICD Pipeline

Cloud-Deployed Bookstore is a full-stack application deployed using cloud infrastructure and automated CI/CD workflows.
The project demonstrates containerized application deployment using AWS services, including Docker, Amazon ECR, and Amazon ECS, along with a CI/CD pipeline to automate builds and deployments.
I was responsible for designing the cloud architecture and implementing the deployment pipeline, ensuring reliable and consistent application updates.
Through this project, I developed practical experience in cloud computing, containerization, and DevOps practices.

### Cloud Kitchen

Cloud Kitchen is a full-stack capstone project focused on simplifying meal planning using available ingredients. Built in collaboration with a teammate, the application allows users to manage a digital pantry and generate recipes based on what they have on hand.

I was responsible for the frontend development, where I designed and implemented an intuitive user interface using Flutter, integrated API services, and ensured smooth data flow between the client and backend. This project strengthened my skills in state management, API integration, and user-focused design.
