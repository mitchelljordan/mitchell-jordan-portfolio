# mitchell-jordan-portfolio

Software development portfolio focused on full-stack applications, cloud architecture, and backend system design.

This repository showcases projects developed throughout my time in Durham College’s Computer Programming and Analysis program. My work emphasizes building scalable systems, designing relational data models, and deploying applications using modern cloud and DevOps practices.

Across these projects, I have worked with REST APIs, containerized environments, CI/CD pipelines, and serverless infrastructure. I focus on creating reliable backend systems, integrating frontend applications, and maintaining data consistency across complex workflows.

Each project represents a stage in my development, from foundational object-oriented programming to production-style cloud deployments, demonstrating both technical growth and practical system design skills.


### Bakery Buddy

Bakery Buddy is a full-stack inventory and production management system designed to model real-world bakery operations.

The application manages ingredient tracking, recipe relationships, and batch production workflows while maintaining accurate inventory levels in real time. I contributed to system design and backend development, implementing relational schemas, transactional logic, and REST API endpoints to ensure data consistency across production and sales processes.

This project strengthened my understanding of database design, state management, and building reliable transactional systems.

### Cloud-Deployed-Bookstore with CICD Pipeline

Cloud-Deployed Bookstore is a full-stack application demonstrating containerized deployment and automated CI/CD workflows.

The system is deployed using AWS services, including Docker, Amazon ECR, and Amazon ECS, with a pipeline that automates build, test, and deployment stages. I designed the cloud architecture and implemented the CI/CD process to ensure consistent and reliable releases.

This project highlights my experience with cloud infrastructure, containerization, and DevOps practices.

### Cloud Kitchen

Cloud Kitchen is a full-stack capstone project focused on simplifying meal planning through ingredient-based recipe generation.

The application allows users to manage a digital pantry and generate meals based on available ingredients. I led frontend development using Flutter, building a responsive UI, integrating backend APIs, and managing application state.

This project strengthened my skills in frontend architecture, API integration, and user-focused design within a full-stack environment.

### DC Registration App

The DC Registration App is an object-oriented system designed to simulate a college course registration process.

Built as part of an OOP-focused course, the project applies core principles such as inheritance, encapsulation, polymorphism, and abstraction to manage students, courses, and registration logic.

This project established my foundation in modular design, code organization, and debugging structured systems.

### Car Service Marketplace

Car Service Marketplace is a serverless web application that connects users with local car service providers.

Built using AWS services including S3, API Gateway, Lambda, and DynamoDB, the system is designed for scalability, cost efficiency, and low operational overhead. I worked on designing and implementing the cloud architecture and backend logic within a serverless environment.

This project highlights my ability to build and deploy distributed systems using modern cloud-native patterns.
