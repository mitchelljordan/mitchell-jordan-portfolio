# Bakery Buddy

Bakery Buddy is a full-stack inventory and production management system designed to replicate real-world bakery operations. The system manages ingredients, recipes, and batch production while ensuring inventory remains accurate and consistent.

The application allows users to:
- Track ingredient inventory levels
- Manage recipes and their ingredient requirements
- Execute batch production processes that automatically update inventory
- Maintain consistency across production and sales workflows
I played a key role in the system’s backend and database design. My contributions included:
- Designing relational database schemas in PostgreSQL
- Implementing stored procedures and business logic for inventory updates
- Handling transactional operations to ensure data integrity during batch production
- Building and validating API endpoints for inventory and production workflows

This project significantly strengthened my skills in backend development, database design, and transactional system logic, particularly in ensuring consistency across complex, state-dependent operations.
